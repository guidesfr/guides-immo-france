Questions fréquentes

Qu'est-ce que la base DVF ?

DVF (Demandes de Valeurs Foncières) est une base de données publique qui recense toutes les transactions immobilières en France depuis 2014. Elle contient le prix de vente, la surface, le type de bien et l'adresse exacte de chaque transaction.

Les données DVF sont-elles fiables ?

Oui, les données DVF sont les prix réels enregistrés chez les notaires. C'est la source la plus fiable, contrairement aux estimations des sites d'annonces qui se basent sur les prix demandés (souvent surévalués de 5-15 %).
